# NHL Manager

Мобильная игра на Unity в жанре хоккейного менеджера.

Игрок становится генеральным менеджером клуба NHL и отвечает за:

- состав команды;
- контракты;
- обмены;
- развитие игроков;
- драфт;
- сезон;
- плей-офф.
