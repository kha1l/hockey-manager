# AGENTS.md

## Проект

nhl-manager — мобильная игра на Unity в жанре хоккейного менеджера.

Игрок управляет клубом NHL, составом, контрактами, обменами, драфтом, финансами и развитием игроков.

Проект должен быть спроектирован так, чтобы поддерживать:

- реальные команды NHL;
- реальные составы NHL;
- реальные сезоны;
- реальные статистические данные;
- реальные логотипы команд;
- реальные фотографии игроков.

На этапе разработки допускается использование временных данных и заглушек.
